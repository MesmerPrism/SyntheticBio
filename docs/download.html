Download & Install

The recommended path is the guided setup bootstrapper, which prompts for admin rights, trusts the preview certificate, and opens App Installer automatically.

Research Preview

This installer is meant for research and developer use. It is self-signed, not backed by a public CA, so Windows still requires an admin-approved trust step. The guided setup helper performs that trust step for you and then opens App Installer.

Fast path

01

Run guided setup

Download SyntheticBio-Preview-Setup.exe from the latest release and open it like a normal installer helper.

02

Accept the admin prompt

Windows asks for elevation because the helper needs to trust the preview certificate in Local Machine > Trusted People.

03

Let it trust the cert and open App Installer

The helper downloads the latest SyntheticBio.cer, imports it into the machine trust store, and opens the published SyntheticBio.appinstaller.

04

Finish in App Installer

App Installer should now show the package as trusted. Complete the install and launch SyntheticBio from the Start menu.

The helper still requires admin approval and may still trigger browser or SmartScreen friction because it is not backed by a publicly trusted code-signing certificate. It removes the need to browse certificate stores manually.

Manual fallback

Use this path if the guided helper is blocked by your environment or you prefer to import the certificate yourself.

01

Download the certificate

Open SyntheticBio.cer from the latest release.

02

Install to the machine store

Choose Install Certificate, select Local Machine, and place it in Trusted People.

03

Open the appinstaller

After the certificate is trusted, open SyntheticBio.appinstaller to finish the install.

Requirements

Windows

Windows 10 version 19041 or later.

Companion use

SyntheticBio is most useful when paired with the PolarH10 app, but the exporter and benchmark paths also work on their own.

Default export location

Installed builds default fixture export to Documents\SyntheticBio\Fixtures.
